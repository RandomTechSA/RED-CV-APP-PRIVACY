# RED-CV-APP-PRIVACY
Complete Android app with document scanning, OCR text recognition, PDF generation, notepad functionality, Cover Letter and CV/resume builder

# REDCV - Privacy Policy

## Data Collection
- **Document Scanning:** Images processed locally on device
- **OCR Processing:** Text recognition performed locally
- **Notepad:** Notes stored locally on device only
- **CV Analysis:** Processing done locally, no cloud uploads

## Data Storage
- All user data stored locally on device
- No data transmitted to external servers
- No third-party analytics or tracking

## Permissions Required
- **Camera:** For document scanning
- **Storage:** For saving PDFs and notes
- **Internet:** Only for app updates (optional)

## Data Sharing
- We do not share user data with third parties
- No data sold to advertisers
- No user profiling or tracking

## Security
- Local encryption for sensitive documents
- Secure keystore for app signing
- No network transmissions of user content

## User Rights
- Full control over your data
- Right to delete all app data
- Right to export your documents
